# PROTOKOL SMOKE TEST HARIAN (PSA Business Suite)

Wajib dilakukan setiap pagi oleh Kasir sebelum transaksi pertama dimulai.

### 1. Kesiapan Perangkat Keras
- [ ] **Printer Termal:** Lakukan cetak "Test Page" atau pastikan lampu indikator berwarna hijau. Jika menggunakan USB, cabut dan pasang kembali jika tidak terdeteksi.
- [ ] **Laci Kasir:** Coba buka manual dengan kunci, lalu tes buka otomatis via aplikasi (Menu Settings > Test Drawer).
- [ ] **Scanner Barcode:** Scan sembarang produk di gudang untuk memastikan karakter masuk ke layar tanpa terpotong.
- [ ] **Timbangan Digital:** Kalibrasi ke angka 0.00 sebelum menimbang perhiasan pelanggan.

### 2. Kesiapan Aplikasi
- [ ] **Koneksi Lokal:** Buka aplikasi. Jika muncul "Aplikasi Siap (Offline-First)", lanjutkan.
- [ ] **Status Sinkronisasi:** Cek icon awan di pojok kanan atas. Jika berwarna merah, pastikan modem menyala (namun tetap bisa jualan).
- [ ] **Shift Baru:** Pastikan sudah melakukan "Open Shift" dengan modal awal yang benar.

### 3. Keamanan
- [ ] **Cek Audit Log:** Pastikan tidak ada aktivitas mencurigakan semalam.
- [ ] **PIN Otorisasi:** Tes PIN Manager Anda sekali untuk memastikan akses otorisasi diskon berjalan.

---
*Jika tes di atas gagal, segera hubungi Manager atau gunakan Nota Manual sementara.*
